# bssm-oss interface system

- Direction: Utility & Function mixed with experimental OSS editor energy
- Palette: bright neutral background, electric blue primary, signal orange secondary, lime highlight
- Depth: soft layered elevation with clear canvas / inspector separation
- Corners: rounded but not soft, 14 / 24 / 36 scale
- Spacing: 4px base scale
- Typography: Space Grotesk for display/body, IBM Plex Mono for UI metadata
- Motion: fast-subtle hover lift only
- Layout: sticky global header, left editable canvas, right persistent inspector
- Interaction states: default, hover, selected, dirty, locked, error, live-sync
- Editing cues: selection outline in accent blue, pin markers in accent-to-signal gradient
