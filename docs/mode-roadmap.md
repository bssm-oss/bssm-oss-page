# Mode Roadmap

## 현재 상태

`AI Mode`와 `Code Mode`는 아직 기능 구현이 아닌 셸 단계다.

둘 다 같은 목적을 향한다.

- PR 설명보다 더 직접적인 수정 흐름
- 화면과 수정 요청의 거리를 줄이는 인터페이스

## AI Mode

### 지금 있는 것

- 수정 대상 페이지 목록
- 자연어 요청 입력 예시
- 에이전트 제안 결과 패널 구조

### 이후 붙을 것

1. 페이지 또는 섹션 선택기
2. 선택 대상을 기준으로 한 agent task 생성
3. 변경 요약과 적용 가능 diff
4. 적용 전후 비교와 승인 흐름

## Code Mode

### 지금 있는 것

- 수정 가능한 영역 목록
- 연결될 코드 미리보기 예시
- 화면과 코드의 직접 연결 설명

### 이후 붙을 것

1. 화면 영역과 컴포넌트/파일 매핑
2. 선택 영역 기준 코드 범위 표시
3. 편집 후 미리보기 즉시 반영
4. 안전한 되돌리기와 변경 기록

## 두 모드가 만나야 하는 지점

장기적으로는 아래 흐름이 중요하다.

- AI Mode가 제안한 변경을 Code Mode에서 바로 검토
- Code Mode에서 수정한 내용을 다시 AI가 설명
- 같은 화면 조각을 기준으로 말과 코드가 연결
